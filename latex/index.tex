This Arduino library prints memory usage on the Serial port.



\subsection*{Features}

This library provides\+:
\begin{DoxyItemize}
\item Print memory usage on the serial console
\item Get stack and heap size
\item Get .data and .bss sections size
\end{DoxyItemize}
